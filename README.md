# 🔍 Network Reconnaissance using Nmap

## 📌 Overview
This project demonstrates network scanning and reconnaissance using Kali Linux and Nmap to identify active hosts, open ports, and services.

---

## 🎯 Objectives
- Discover active devices on a network  
- Identify open ports  
- Detect running services  
- Analyze potential security risks  

---

## 🧰 Tools Used
- Kali Linux  
- Nmap  

---

## ⚙️ Commands Used

### Host Discovery
nmap -sn 192.168.1.0/24

### TCP SYN Scan
nmap -sS 192.168.1.1

### Service Detection
nmap -sV 192.168.1.1

---

## 📊 Findings
- Active hosts discovered  
- Open ports (22, 80)  
- Services identified  

---

## 🔐 Security Recommendations
- Close unused ports  
- Configure firewall  
- Update services  
- Use strong authentication
